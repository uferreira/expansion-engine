# The Expansion Engine

The public website for the Expansion Engine, an independent research project exploring how low-temperature thermal energy can be converted into useful mechanical and electrical power.

The project was conceived and is led by physicist **Victor de Amorim d’Ávila**, with collaboration from oceanographer **Uggo Pinho**.

## Website

This repository contains a dependency-free static website built for GitHub Pages.

To preview it locally, open `index.html` in a browser or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Project status

The Expansion Engine is under active research, prototype development, testing, and refinement. Detailed engineering information remains proprietary.
